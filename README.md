# DevStore

**DevStore** is a simple and modern e-commerce web application built with **Next.js (latest version)**, designed to follow the best practices of a **server-side rendered (SSR)** web application. This project also includes **end-to-end testing with Cypress**, making it a solid example of quality and maintainable React/Next.js development.

---

## 📌 Features

- 🛍️ Simple e-commerce application structure  
- ⚡ Built with **Next.js** (SSR optimized)  
- 🧪 Automated **Cypress** tests included  
- 💡 Uses best practices for scalable and maintainable code  
- 🧠 TypeScript-based project setup  
- 💻 Fully configured for a modern development workflow  

---

## 🚀 Technologies

This project uses the following technologies:

- **Next.js** — React framework for server-side rendering  
- **TypeScript** — Type safety and improved developer experience  
- **Cypress** — End-to-end testing framework  
- **pnpm** — Fast and efficient package manager  
- **ESLint** — Code quality and linting  
- **PostCSS** — CSS transformation tool

## 🧪 Running Tests

The project is configured with Cypress for testing. To run tests locally:

```bash
# Install dependencies
pnpm install

# Run Cypress test suite
pnpm run cypress
```

## 🛠️ Getting Started

To start the development server:

```bash
# Install dependencies
pnpm install

# Start the dev server
pnpm run dev
```

Then open your browser and go to:

```bash
http://localhost:3000
```
